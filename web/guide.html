👤 이름·비밀번호 설정

✅ 가입 완료
  • 전화번호를 입력하면 SMS 인증코드(6자리)가 발송됩니다.
  • 인증코드는 3분간 유효합니다.
  • 인증 완료 후 이름과 비밀번호를 설정하면 가입이 완료됩니다.
  • 비밀번호는 8자 이상, 영문+숫자 조합이 필요합니다.

로그인

가입한 전화번호와 비밀번호로 로그인합니다. 로그인하면 자동으로 대시보드로 이동합니다.

비밀번호 찾기

로그인 화면에서 "비밀번호 찾기"를 누르면 전화번호 인증 후 새 비밀번호를 설정할 수 있습니다.

Step 2

업장 관리

업장 생성

처음 사용 시 업장을 생성합니다. 업장을 생성한 사람은 자동으로 OWNER(소유자) 역할이 됩니다.

  • 업장 이름 — 예: CarAI 정비소
  • 주소 — 예: 서울시 강남구
  • 전화번호 — 예: 02-1234-5678

직원 초대

업장에 직원을 초대하는 두 가지 방법이 있습니다.

방법설명
전화번호로 초대직원의 전화번호를 입력하면 초대 링크가 SMS로 발송됩니다.
직접 초대이미 가입된 사용자를 ID로 직접 초대합니다.

초대받은 사용자는 초대를 수락하면 업장에 합류합니다.

직원 관리

  • 역할 변경 — OWNER가 직원의 역할을 변경할 수 있습니다.
  • 직원 내보내기 — MANAGER 이상이 직원을 내보낼 수 있습니다.
  • 업장 탈퇴 — 본인이 직접 업장을 나갈 수 있습니다.

안전점검표는 차량 점검 시 사용하는 체크리스트입니다. 시스템에서 기본 제공하는 점검표를 업장에 등록하여 사용합니다.

Step 4

접수 및 작업 할당

엑셀 수납 이력 분석

고객의 수납 이력 엑셀 파일을 업로드하면 시스템이 자동으로 분석합니다.

👤

고객 정보 추출

고객명, 전화번호, 차량번호, 차종을 자동 식별합니다.

💳

결제 이력 정리

최근 결제 내역을 날짜순으로 정리합니다.

🔧

정비 항목 요약

항목별 마지막 정비일과 주행거리를 요약합니다.

⚠️

권장 정비 안내

주기 초과 항목을 자동으로 알려줍니다.

작업 할당

분석 결과를 확인한 후, 담당 정비사에게 작업을 할당합니다.

  • 담당 정비사를 선택합니다.
  • 작업 설명을 입력합니다 (예: 엔진오일 교체 및 브레이크 점검).
  • 고객/차량 정보와 수납 이력이 자동으로 첨부됩니다.
MANAGER 이상의 권한이 필요합니다.
Step 5

작업 진행

작업은 다음 상태를 거칩니다.

대기 중 → 작업 중 → 완료 → 소견서 생성 중 → 소견서 완료

작업 시작

대기 중인 작업을 선택하고 사용할 안전점검표를 선택하면 작업이 시작됩니다.

점검 진행 및 임시 저장

  • 점검표의 각 항목을 체크하며 점검을 진행합니다.
  • 중간에 임시 저장하면 나중에 이어서 진행할 수 있습니다.
  • 앱을 종료해도 데이터가 유지됩니다.

작업 완료

모든 점검이 끝나면 "작업 완료"를 눌러 최종 결과를 저장합니다. 완료 후 자동으로 AI 소견서 화면으로 이동합니다.

작업 취소

대기 중 또는 작업 중 상태의 작업은 취소할 수 있습니다.
